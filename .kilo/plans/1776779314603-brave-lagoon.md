## Testing Strategy

### 1. API-Based Test Suite
All tests call `POST /exec` HTTP endpoint - same entry point users use.

### 2. Test Structure
```
tests/
  schema.test.ts          # Validate all 131 node definitions
  fixtures/
    graphs/               # Pre-defined graphs from tests/pipelines/*.nuon
  nodes/
    unit.test.ts          # Test each node type via /exec
    integration.test.ts    # Multi-node chains
```

### 3. What Gets Tested

**Schema validation** (131 nodes):
- Each node has required fields: name, category, ports, params
- Ports have valid types
- Params have valid options

**Unit tests** (all nodes):
- Call `/exec` with minimal graph
- Assert result matches expected output

**Integration tests**:
- Multi-node pipelines
- Extension nodes (htmd, feargreed, hn, etc.)

### 4. Test Runner
Use `bun test` - already in project, fast.

## Implementation Steps

1. Create `tests/schema.test.ts` - validates all node definitions
2. Create `tests/fixtures/graphs/` - copy existing .nuon files as JSON
3. Create `tests/nodes/unit.test.ts` - loop all nodes, exec, assert
4. Create `tests/nodes/integration.test.ts` - chain tests
5. Remove old `tests/` and `demos/`
6. Run `bun test` to verify

All 131 tested via schema validation. Core + extension tested via execution.

## Test Examples

```typescript
// schema.test.ts - validates node definitions
test('all nodes have required fields', () => {
  const defs = await fetch('/defs')
  for (const node of defs) {
    assert(node.name)
    assert(node.category)
    assert(node.ports?.inputs)
    assert(node.ports?.outputs)
  }
})

// unit.test.ts - tests each node
test('math node: 10 + 5 = 15', async () => {
  const res = await exec({
    nodes: [
      { id: 'a', type: 'const', params: { value: '10' } },
      { id: 'b', type: 'math', params: { op: '+', operand: '5' } },
      { id: 'out', type: 'return', params: {} }
    ],
    edges: [
      { from: 'a', from_port: 'output', to: 'b', to_port: 'input' },
      { from: 'b', from_port: 'output', to: 'out', to_port: 'input' }
    ]
  })
  assertEquals(res.result, 15)
})
```
