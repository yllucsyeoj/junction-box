# Plan: `/visualise` Endpoint

## Overview
Add a `GET /visualise/:alias` endpoint that renders a saved patch as an ASCII flowchart diagram using Mermaid + ascii-mermaid.

---

## Scope

### In Scope
- `GET /visualise/:alias` — retrieve patch by alias, convert graph to Mermaid, render with ascii-mermaid
- Return the ASCII diagram as plain text response
- Error handling for missing patches and invalid graphs

### Out of Scope
- POST /visualise with inline graph (user specified only saved patches)
- Interactive/dynamic layout algorithms
- SVG/HTML rendering (only ASCII per user request)
- Gallery/index view of all patches

---

## Dependencies

**mermaid-ascii** is a Go binary, not a JS package. We shell out to it as a subprocess.

Binary management approach:
- Download `mermaid-ascii` binary on first run (or as part of setup)
- Cache it locally (e.g., `data/bin/mermaid-ascii`)
- Shell out via ` Bun.spawn` or `execSync` to render mermaid → ASCII

No npm dependencies needed for rendering.
